#HOW TO USE THESE SCRIPTS

Access the script, and each of them should have a comment/usage section. 
Each should have a snippet of code that you can copy/paste to run it standalone.

**Note:** Please do not copy and paste the **full** script code into a notepad and save it for future use. This will result in you not getting bug fixes and new features on each script as i update them. Each script's #usage snippet should mostly be downloading the freshest code from this github repo, and then maybe executing it with variables and/or function calls.

Important things of note: Many of these will have plenty of errors. I try to notify you when an error might occur that should be ignored, however also feel free to teams message David with any errors.

A note on confidentiality: All of these scripts have been sanitized to the best of my knowledge of any identifying information. Should you discover something i missed, please inform me IMMEDIATELY. There should be no client names, our company name should not be written out anywhere, no serial keys, no domain names, no emails, etc.

Lastly, for full-pc setup scripts and scrips with API keys and other sensitive information: 
I also have a secondary private repo with those scripts. Please teams message David to request access.
